
# 🎓 Student Performance Prediction System

## 📌 Problem Statement

Educational institutions often need to identify students who may require additional academic support. This project uses Machine Learning to predict a student's final academic performance based on factors such as study hours, attendance, assignments completed, and previous scores.

---

## 📊 Dataset Information

The dataset contains student academic records with the following features:

* Study Hours
* Attendance (%)
* Assignments Completed
* Previous Score
* Final Marks (Target Variable)

The dataset was cleaned and preprocessed before training the machine learning model.

---

## 🤖 Machine Learning Algorithm

**Linear Regression**

Linear Regression was used to learn the relationship between student attributes and final marks, enabling the prediction of future student performance.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Gradio
* Hugging Face Spaces

---

## 📈 Model Performance

### Evaluation Metrics

* R² Score: **0.93**
* Model: Linear Regression

The model achieved strong predictive performance, explaining approximately 93% of the variance in student final marks.
